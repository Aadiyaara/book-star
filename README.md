# Bookstar app for Pratilipi

by - Aditya Sharma

## FEATURES

- Make an account
- View Library
- Helpful filter to guide you through the library
- Publish a book
- Manage your books
- View Book Summaries
- et users see your updated book in the front on Last Updated List (Last list in the UI)

## Project Setup

### CLIENT

1. `npm install`
2. `npm start`

### SERVER

1. `npm install`
2. `npm run dev`
